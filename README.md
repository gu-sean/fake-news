# Fake-News

뉴스 크롤링 및 모델 기반 가짜뉴스 판별 시스템


## 1. 프로젝트 개요

딥러닝 기반 자연어 처리(NLP)를 활용해 가짜 뉴스를 실시간으로 탐지하고, 정보의 신뢰도를 확보하여 사회적 혼란을 방지합니다.


## 2. 주요 특징

### DVL (Dynamic Verification Layer)
- 통계 왜곡
- 인과 오류
- 감정 자극
- 출처 불명
- 이미지 불일치

위 5가지 요소를 기반으로 뉴스 신뢰도를 정량적으로 분석합니다.

### Hybrid Architecture
`KLUE-RoBERTa`의 문맥 이해 능력과  
`DVL`의 통계 기반 검증 시스템을 결합하여  
텍스트 의미 분석 + 정량 검증이 동시에 가능하도록 설계했습니다.
